## Алгоритм генерации данных
Обозначим через $\mathcal{G}(F, G, a, n, z)$ - алгоритм генерации набора данных размера $n$ для модели $Y = aX + \varepsilon$, в которой случайные величины $X$ и $\varepsilon$ являются независимыми и имеют распределения $F$ и $G$ соответственно, а также для этого набора данных $l(n) = \left[z \cdot n \right]$, где $z \in [0, 1]$.

Алгоритм генерации.

Положим $n_d = \left[z \cdot n \right]$, $n_i = n - n_d$. Алгоритм $\mathcal{G}(F, G, a, n, z)$ устроен следующим образом:

1. Генерация пар:

- Из распределения $F$ сгенерировать $n_d$ независимых случайных величин $X_j$;
- Из распределения $G$, независимо от полученных на предыдущем шаге $X_j$, сгенерировать $n_d$ независимых случайных величин $\varepsilon_j$;
- Построить выборку размера $n_{dep}$ по формуле $Y_j = aX_j + \varepsilon_j$.

2. Генерация случайных величин без пар:

- Из распределения $F$, независимо от всего остального, сгенерировать $n_i$ независимых случайных величин $\widetilde{X}_j$;
- Из распределения $G$, независимо от всего остального, сгенерировать $n_i$ независимых случайных величин $\widetilde{\varepsilon}_j$;
- Построить выборку размера $n_i$ по формуле $Y_j = a\widetilde{X}_j + \widetilde{\varepsilon}_j$;
- Из распределения $F$, независимо от всего остального, сгенерировать $n_i$ независимых случайных величин $X_j$.

3. Объеденить $\{X_i\}$ полученные на шаге (1) и (2) и перемешать случайным образом. Аналогично для $\{ Y_i \}$.
